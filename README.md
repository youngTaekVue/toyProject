# TOYPROJECT
  

# 1. 저장소 복제
git clone [프로젝트 저장소 URL]
cd [프로젝트 폴더명]

# 2. 프론트엔드 의존성 설치 (루트 디렉토리)
npm install # 또는 yarn install
# [필요하다면: cd frontend 폴더로 이동 후 설치 명령 추가]

# 3. 백엔드 의존성 설치
cd backend
npm install # 또는 yarn install


001.swapSample	⚙️ UI/UX	두 요소의 상태나 내용을 Swap/Toggle 하는 기능 구현 예제.

002.customElement	🌐 프론트엔드	Web Components를 활용한 사용자 정의 HTML 엘리먼트 생성 사례.

003.lazy-load-list	⚡ 성능	스크롤에 따라 데이터를 비동기로 로드하는 무한 스크롤(Lazy Loading) 목록 구현.

004.thermometer	📊 데이터 시각화	데이터를 아날로그 형태로 표현하는 온도계 UI/UX 컴포넌트 데모.

005.calendar	📅 UI/UX	날짜 선택, 표시 기능 등을 포함한 달력 컴포넌트 구현.

006.weather	📡 API 연동	외부 날씨 API를 호출하여 데이터를 표시하는 위젯 데모.

backend	🖥️ 서버/API	프론트엔드 데모들을 위한 백엔드 API 서버 코드.

svgAnimation	🎨 애니메이션	**SVG(Scalable Vector Graphics)**를 활용한 동적이고 인터랙티브한 애니메이션 코드.
